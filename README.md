# Rail Slider 2D - Master Edition
- Rail Slider 2Dは、スライドパズルを操作して列車をゴールまで導くアクションパズルゲームです。1980年代の名作「チクタクバンバン」にインスパイアされ、現代のWeb技術（HTML5 / JavaScript / Web Audio API）で再構築されました。

## 🎮 プレイ方法
1. **[START] ボタン**を押して列車を動かします。
2. 空きマスを利用してパネルを**スライド**させ、レールのルートを繋げます。
3. 列車が途切れたレールや障害物にぶつかるとクラッシュします。
4. 最上段の**[GOAL]**マスに到達すればステージクリアです！

## ✨ 特徴
- **3つの難易度レベル**: パズルゾーンの広さが変化します。
- **リアルな演出**:
  - Web Audio APIによる汽笛、走行音（チクタク音）、拍手喝采。
  - 蒸気、パーティクル、コミカルな揺れアニメーション。
- **クリア報酬**: 各レベルのクリア回数を記憶。特定の回数クリアで隠しメッセージが出現。
- **デモモード**: 15秒間操作がないと、自動的にデモンストレーション走行が始まります。
- **カスタマイズ**: 5種類の洗練されたテーマカラーを用意。

## 🛠 技術スタック
- 純粋なHTML/CSS/JS: フレームワークを一切使用せず、シングルファイルで動作します。
- **Web Audio API**: 音声合成によるBGM/SE生成。
- **ベクトルグラフィックス (SVG)**: 美しく拡大縮小されるレール表示。

# 🛠 アレンジモードのご案内 (Arrange Mode Guide)

Rail Slider 2D V1.0 では、プレイヤーがゲーム体験を自分好みにカスタマイズできる**「アレンジモード」**を搭載しています。このモードを使えば、クリア後の達成感をさらに特別なものに変えることができます。
- ※但しこれまでにクリアした回数（王冠の数）もリセットされます。ご了承ください。
## 🌟 アレンジモードとは？
特定の回数で表示される「メッセージ」を、あなたオリジナルのメッセージに書き換え、クリアした時に表示させる特別なモードです。
- 自分への励ましに: 難しいパズルを解いた自分へのメッセージをセット。
- サプライズに: 友人や家族にプレイしてもらい、クリアした瞬間に秘密のメッセージを表示。
- 漢字変換・多言語対応: 日本語（漢字・ひらがな・カタカナ）はもちろん、英文なども自由に入力可能です。
-
## 🚀 使い方
1. **設定メニューを開く**:ゲーム画面下の「▼ SETTINGS」をタップします。
2. **アレンジボタンを選択**: 黄色の「アレンジ」ボタンを押して専用メニューを開きます。
3. **メッセージを入力**:テキストエリアに好きなメッセージを自由に入力してください（最大150文字）。
4. **レベルを選択**: メッセージを表示させたい難易度（Level 1〜3）を選びます。アレンジモード中に設定メニューから切り替えることも可能。
5. **保存して開始**: 「保存してアレンジモード開始」ボタンを押します。

## 💡 モード中の変化
- 画面左上のレベル表示が 「アレンジ」 に切り替わります。
- 選択したレベルをクリアする度に、設定したオリジナルメッセージが必ず表示されます。

## 🧹 解除方法（通常モードに戻す）
アレンジモードを終了したい時は、以下の操作を行ってください。
1. 「▼ SETTINGS」メニューを開きます。
2. 赤色の 「ALL RESET」 ボタンを押します。
　- ※これによりアレンジモードが解除され、全てが初期状態にリセットされます。

## 📝 仕様と制限
- **文字数制限**: 最大150文字まで。スマートフォンの画面でも読みやすい最適なボリュームに設定されています。
- **リセット**: 「ALL RESET」を実行するまで、アレンジ設定は維持されます。
- **入力のヒント**: メッセージ入力欄をタップすると自動で空白になります。新しく書き直す際に便利です。
