# Rail Slider 2D - Master Edition
- Rail Slider 2Dは、スライドパズルを操作して列車をゴールまで導くアクションパズルゲームです。1980年代の名作「チクタクバンバン」にインスパイアされ、現代のWeb技術（HTML5 / JavaScript / Web Audio API）で再構築されました。
## 🎮 プレイ方法
1. **[START] ボタン**を押して列車を動かします。
2. 空きマスを利用してパネルを**スライド**させ、レールのルートを繋げます。
3. 列車が途切れたレールや障害物にぶつかるとクラッシュします。
4. 最上段の**[GOAL]**マスの中心に到達すればステージクリアです！

## ✨ 特徴
- **3つの難易度レベル**: パズルゾーンの広さが変化します。
- **リアルな演出**:
  - Web Audio APIによる汽笛、走行音（チクタク音）、拍手喝采。
  - 蒸気、パーティクル、コミカルな揺れアニメーション。
- **クリア報酬**: 各レベルのクリア回数を記憶。特定の回数クリアで隠しメッセージが出現。
- **デモモード**: 15秒間操作がないと、自動的にデモンストレーション走行が始まります。
- **カスタマイズ**: 5種類の洗練されたテーマカラーを用意。

## 🛠 技術スタック
- 純粋なHTML/CSS/JS: フレームワークを一切使用せず、シングルファイルで動作します。
- **Web Audio API**: 音声合成によるBGM/SE生成。
- **ベクトルグラフィックス (SVG)**: 美しく拡大縮小されるレール表示。
